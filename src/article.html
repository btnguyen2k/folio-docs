Introduction

What folio-docs is, who it is for, and what you can build with it.

Updated 2026-07-01 · 3 min read

overview concepts

Welcome to Folio Docs — a lightweight template for building web-based technical documentation and user guides. It ships as plain HTML, CSS and JavaScript styled with Tailwind CSS, so there is nothing to compile before you can open it in a browser.

Why folio-docs

  • Zero-framework: pure HTML/CSS with a touch of vanilla JS.
  • Content and layout are cleanly separated.
  • Light and dark themes out of the box.

Continue with Installation to set things up.
